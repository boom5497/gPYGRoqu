# 项目简介

## 核心功能
- 基于YOLOv8的骑行头盔目标检测系统，可识别图片中的头盔(head)和头部(head)两类目标
- 支持多种输入方式：单张图片、图片文件夹、视频文件及本地摄像头
- 提供实时显示功能：目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 可选择显示标签和原图
- 提供批量检测功能，可查看每一张检测结果
- 支持检测结果保存
- 包含6000张标注数据集，涵盖['head', 'helmet']两类

## 适用场景
- 骑行安全监测系统
- 工业安全防护检测
- 安防监控系统
- 交通违规检测
- 安全生产管理

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，集成opencv-python、PyQt5、ultralytics、torch1.9等框架
- 提供完整UI界面，包括源文件、图标等
- 包含完整程序文件、数据集图片及项目运行教程
- 已训练好模型，配置环境后可直接使用
